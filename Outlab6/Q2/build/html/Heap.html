Heapify(root)[source]¶

This function makes a heap out of a tree whose left and right subtrees are heaps.

Parameters:

root – The root of the heap.
Returns:

None
Example:
>>> from DSA import Heap
>>> x = Heap(3)
>>> x.H = [100, 2, 3]
>>> x.n = 3
>>> x.Heapify(0)
>>> x.H
[2, 100, 3]
