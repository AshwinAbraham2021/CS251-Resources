Doubly Linked List¶

class DSA.DoublyLinkedListNode(data)[source]¶

Bases: object

This class represents each node in a doubly linked list. Each object has attributes

  • data (The data stored in the node)
  • prev (The previous node in the list)
  • next (The next node in the list)
The list of available functions are
  • __init__(data) (A constructor)
  • __str__() (converts the node object to a string)
__init__(data)[source]¶

This is a constructor for the Node object.

Parameters:data – Represents the data stored in the node.

prev and next are initialized to None.

Returns:

None
Example:
>>> from DSA import DoublyLinkedListNode
>>> x = DoublyLinkedListNode(4)
>>> x.data
4
__str__()[source]¶

This converts the node object to a string. This is done by converting the data stored in it to a string.

Returns:

string

>>> x = DoublyLinkedListNode(4)
